# GSP-AllianRyanBruce

	Our app is a Super-Smash Bros styled platform fighter designed around accumulating damage and knockback on enemies, with varied movement options both on the ground and in the air. 

Characters:
Characters are very customizable due to an expandable and modular base script to allow each character to feel unique and balanced.
Each character will have seven grounded basic attacks: 
4 basic attacks:  neutral attack, forward attack, up attack and down attack 
3 power attacks: forward, up and down
Each character will have 5 aerial attacks: neutral, forward, back, down and up
Each character has a universal shield that protects them from attacks but depletes when damaged and over time
Each character has a grab that is weak to attacks but can pull enemies out of shield
Each character has a cooldown-based dodge that gives temporary invincibility and short range burst movement in exchange for losing momentum afterwards
Each character has 4 directional specials that are more unique abilities 
Up special, down special, neutral special and side special

The game should be easily accessible to anyone and understandable to casual and competitive players.
